A music display built with a ESP32-C3 and a ST7735 1.8" screen that shows what you're currently listening to, and works with any music app (Apple Music, Spotify, YouTube Music, anything as long as it´s Last.fm supported) through Last.fm scrobbling.

still work in progress.

## Why I Made This Project

I built this project primarily to practice CAD, as I prefer learning by doing projects and figuring things out as I go. Through this build, I learned how to create and separate parts in a single Part Studio, join imports in an Assembly, and other fundamental 3D modeling concepts. It was excellent practice for getting familiar with 3D modeling thinking.

Additionally, I love music and thought this would be a great way for people in real life to see what I'm listening to in real-time, regardless of which music app I'm using.
